Source code for [my Offical Portfolio Website](https://www.kelvinkamau.app)

#Dependencies
* [GulpJS](https://gulpjs.com)
* [Ruby](https://)

## Usage
* Run ```gem install jekyll sass bundler jekyll-minifier jekyll sitemap```
* Run ```npm install```
* Run ```npm install --save-dev gulp-install```

#### Development
* Run ```gulp serve```

#### Production
* Run ```gulp build```
